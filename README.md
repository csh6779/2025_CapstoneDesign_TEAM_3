- 반도체는 대한민국의 강력한 수출 품목이며 기술의 발전에 따라 크기 또한 매우 작아지고 있음

- 반도체와 같은 매우 작은 요소들을 검사하기 위한 사진의 크기는 매우 큼 (수십 ~ 수백 메가 픽셀 필요)

- 위와 같은 사진을 평범한 로컬 환경에서 일반적인 실행을 하게 되면 속도가 매우 느림

- 사진을 타일의 형식으로 해체하여 저장하고 사용자가 확대/축소할 떄 현재 화면에 필요한 타일만 로딩해서 빠르게 보여주는 방식

- 2D 이미지를 빠르게 볼 수 있게 할 것이며, 3D 볼륨 데이터를 탐색 및 분석할 수 있게 할 것
